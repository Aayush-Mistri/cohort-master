### Cohort Community Platform

A place where developers don't just learn together — they build together.

### What is this?

This project is the digital home of our cohort — a real-time community platform designed for collaboration, discussion, events, and shared growth.

Think of it as a mix of Discord + LinkedIn + a campus community board, built entirely by developers from the cohort.

Current Features

Live

Community feed

Share updates, achievements, and announcements

Real-time chat

Event management

Create events and manage RSVPs

Scheduled messages

Plan announcements in advance

Responsive UI

Optimized for desktop and mobile

Authentication system

Secure user login and registration

### Getting Started

Installation

5 steps

1

Clone the repository

bash

git clone [https://github.com/your-cohort/community-platform.git](https://github.com/your-cohort/community-platform.git)

2

Install dependencies

bash

cd community-platform

npm install

3

Add environment variables

env

MONGODB_URI=your_mongodb_uri

JWT_SECRET=your_secret

PORT=5000

4

Run the backend

bash

npm run server

5

Run the frontend

bash

npm run dev
